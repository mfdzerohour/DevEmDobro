Declarando Variáveis

Existem três tipos de variáveis var, const e let.

var

Variável var, é usada para definir uma variável que pode ter seu valor alterado.

var nomePessoa = "Marcelo"; o nome informado ali pode sofrer alterações no código, onde ele funciona em global, se o mesmo estiver dentro de um bloco de códigos o valor dele, vai valer para os elementos fora dele (global).

const

Variável const, seu valor não pode ser alterado me momento nenhum, é um valor absoluto durante a execução do sistema, onde ele funciona em bloco, se estiver dentro do bloco ele não tem visualização global apenas local.

let

Variável let, funcionamento igual ao var onde ele funciona em bloco, se estiver dentro do bloco ele não tem visualização global apenas local.
