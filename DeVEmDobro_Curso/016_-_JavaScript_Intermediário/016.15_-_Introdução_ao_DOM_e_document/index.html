Introdução ao DOM e document

Agora vamos aprender o que é o DOM (Document Object Mode) e como acessar ele pela palavra-chave document. Com ele, é possível pegar os elementos no HTML do nosso documento e manipular para deixar nossas páginas mais interessantes e dinâmicas.

Apenas foi mostrado que digitando document, ele vai mostrar as propriedades da página WEB, seria o inicio uma "raiz" como uma arvores e todos os componentes da página WEB.
