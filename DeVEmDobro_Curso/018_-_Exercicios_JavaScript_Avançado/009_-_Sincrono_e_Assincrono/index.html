Síncrono e Assíncrono

Síncrono => Quando ele segue um fluxo linear, de cima para baixo, só executa uma instrução do JS quando a anterior é executada.

Assíncrono => Ele executa de forma a não esperar que a instrução anterior terina/seja executada, é o caso do setTimeout, que ele não vai "esperar" que seja executada seu interior, executando o que estiver externo.
