Gerenciando pacotes com NPM

Nesta aula avançada sobre JavaScript, você aprenderá como gerenciar pacotes com o NPM (Node Package Manager). O NPM é uma ferramenta essencial para o desenvolvimento de projetos JavaScript, permitindo que você instale, atualize e gerencie facilmente as dependências do seu projeto.

E rode o arquivo no shell porque no navegador da erro:
node.exe .\index.js

Apenas por exemplo nesta aula foi subido para o github a pasta node_modules, em um projeto normal não deve subir a mesma.

Para instalar os arquivos e suas dependencias, deve entrar na pasta do projeto com npm e rodar npm install, ele vai instalar todas as dependencias do projeto e instalar os arquivos.
