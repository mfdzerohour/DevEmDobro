Exportação default e exportação nomeada

Nesta aula, você explorará os conceitos de exportação de módulos usando as declarações "export default" e "export nomeado". Essas são duas formas poderosas de compartilhar código entre diferentes arquivos JavaScript.

Primeiramente, você aprenderá sobre a exportação padrão ("export default"), que permite exportar um único valor, função ou objeto como o valor padrão de um módulo. Você entenderá como usar essa sintaxe para exportar e importar módulos de forma mais concisa.

Em seguida, você será apresentado à exportação nomeada ("export nomeado"), que permite exportar vários valores, funções ou objetos de um módulo. Você aprenderá como usar essa sintaxe para exportar várias entidades específicas e como importá-las corretamente em outros arquivos JavaScript.
