Como manipular os elementos do DOM

Como já aprendemos a selecionar os elementos, agora vamos manipular através do JS. Ela é importante por que várias das interações que nós vemos nos sites hoje me dia são feitas a partir desses métodos.

IMPORTANTE: A partir do minuto 12:10 é importante que use o acento grave (`) para o funcionamento do código corretamente. Não se deve usar aspas simples e nem duplas. Uma das formas de se verificar se deu certo é que quando digitarmos ${} ele ficará colorido
