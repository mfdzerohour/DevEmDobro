Listas no React e a prop "key"

Nesta aula, você aprenderá sobre um conceito importante: listas no React e a prop "key". Listas são elementos essenciais em muitas aplicações React, e entender como trabalhar com elas é fundamental para desenvolver interfaces dinâmicas e eficientes.

Você aprenderá como criar e renderizar listas no React, utilizando o método map para percorrer e exibir os elementos de um array. Além disso, você será apresentado à propriedade "key", que deve ser atribuída a cada item da lista para ajudar o React a identificar e atualizar corretamente os componentes.
