Rotas no React

Nesta aula sobre rotas no React, você aprenderá como implementar um sistema de roteamento eficiente em sua página. As rotas são fundamentais para criar aplicações web complexas e interativas, permitindo a navegação entre diferentes páginas e componentes.

Lembrando que durante as gravações foi utilizado a versão 5 do React Router
