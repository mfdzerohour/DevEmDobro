Lifecycle ou ciclos de vida no React

Clico de vida

O componente react vai ter 4 fases

  1. Inicialização:
    • na fase de inicialização, como o nome já diz, o compoentent React se prepara para sua inicialização, geralmente ele vai configurar os estados iniciais, usando o objeto state o porps padrões se houverem

  2. Montagem:

    3. Aqui ele esta pronot para ser montado no DOM do navegador. Nesta fase o componente vai seguir o seguinte fluxo de método:

    • Construtor()
    • getDerivedStateFromProps() *Ele chama um metodo antes do render*
    • render()
    • componentDidMount()

  3. Atualização:

    4. Esta fase começa quando o component React já nasceu no navegador e cresce recebendo novas atualizações. Os métodos que são chamados nesssa fase são:

    1. getDerivedStateFromProps()
    2. shouldComponentUpdate()
    3. render();
    4. getSnapshotBeforeUpdate()
    5. componentDidUpdate()

    6. Desmontagem:

      7. Nessa fase, o componente não é mais necessário e vai ser desmontado/retirado do DOM. O método que se chama nesta fase é:

      • componentWillUnmount():

    Observações:

    Nem todos os métodos do ciclo de vida são obrigatórios. Você pode usar apenas os métodos que são relevantes para o seu componente.

    Os métodos do ciclo de vida são chamados automaticamente pelo React em momentos específicos.

    É importante entender o ciclo de vida dos componentes React para criar aplicativos eficientes e escaláveis.
