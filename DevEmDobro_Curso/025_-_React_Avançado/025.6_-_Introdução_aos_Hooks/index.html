Introdução aos Hooks

Hooks

  • Hooks ou gancho em português são uma nova adição ao React que veio na versão 16.8
  • Permitem trabalhar com estado e com ciclos de vida do React sem usar o componente de classe, só dentro de componentes de função.
  • é importante saber que os Hooks não funcionam dentro de classes

Regras

  • Não chame Hooks dentro de loops, condições ou funções aninhadas.
  • Apenas chame Hooks de componentes funcionais. Não chame Hooks de funções JavaScript comuns.
