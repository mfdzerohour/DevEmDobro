TypeScript - Entendendo o TypeScript

TypeScript é uma linguagem de programação fortemente tipada que se baseia em JavaScript, o nome dela já diz Type, de tipo, então ela vai ser muito usada quando queremos tipar nosso código, calma que logo vamos explicar por que tipagem é tão importante.

O TS foi desenvolvido pela Microsoft e é considerado um superset do JS, que seria uma linguagem construída “em cima” do javascript tradicional.
