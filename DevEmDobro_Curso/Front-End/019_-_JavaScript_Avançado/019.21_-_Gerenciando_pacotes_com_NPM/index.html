Gerenciando pacotes com NPM

Nesta aula avançada sobre JavaScript, você aprenderá como gerenciar pacotes com o NPM (Node Package Manager). O NPM é uma ferramenta essencial para o desenvolvimento de projetos JavaScript, permitindo que você instale, atualize e gerencie facilmente as dependências do seu projeto.

Tem que estar na pasta do projeto

Executar os comandos:

  • npm init
  • colocar o nome do projeto
  • Informar o a versão do projeto: Pode deixar o padrão (1.0.0)
  • A descrição do projeto
  • entry point: ele vai pedir o nome, no nosso caso pode deixar o sugerido index.js
  • Test Command: algum comando que queriamos fazer teste, por enquanto deixar em branco na criação
  • git repository: seria o repositorio do git, por euqanto ainda não esta criado, pode deixar em branco
  • keywords: PALAVRA CHAVE
  • Author: colocar o nome do autor
  • Licença por enquanto a padrão
  • Pede a confirmação se tudo está OK
