Utilizando múltiplos componentes (aninhando componentes)

Nesta aula, vamos aprender a como utilizar múltiplos componentes e aninhar componentes para criar interfaces mais complexas.

Durante a aula, você será introduzido aos conceitos de componentização e reutilização de código no React. Você aprenderá como dividir sua interface em componentes menores e independentes, e como aninhá-los para construir interfaces mais complexas e escaláveis.
