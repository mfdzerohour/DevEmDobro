A base do React - O JSX

Nesta aula introdutória, você será apresentado ao conceito de JSX e sua importância no desenvolvimento de aplicações React. O JSX é uma extensão da sintaxe do JavaScript que permite escrever código HTML/XML dentro do JavaScript.

JSX (JavaScript XML) é uma extensão de sintaxe do JavaScript que permite escrever código semelhante ao HTML dentro de um arquivo JavaScript. É usado no React para criar interfaces de usuário

O JSX é semelhante ao HTML, com tags de abertura e fechamento, atributos e elementos aninhados.

O JSX é desenvolvido pela Meta (anteriormente Facebook).

Algumas características do JSX são:

  • Permite misturar código JavaScript com sintaxe de HTML;
  • Permite escrever componentes de interface de usuário em um único arquivo;
  • É mais intuitivo e menos burocrático do que o JavaScript puro;
  • É mais conciso e a maioria dos desenvolvedores React o prefere.

Para converter o JSX em código JavaScript compreensível pelo navegador, é possível usar uma ferramenta como o Babel.
